Elementos no HTML

Elementos no HTML, também chamados de "tags", servem para nós estruturarmos nossa página, colocando títulos, imagens, paragráfos, listas, abreviações e entre outras estruturas.
Como estamos falando da página, incluímos ambas as partes, o <body> e o <head>, ou seja, estas estruturas podem ser tanto um elemento totalmente
visível (como um título), ou até mesmo elementos não vísiveis, utilizados apenas para SEO.

    Elementos visíveis na página

  • <h1>  - Tag utilizada para fazer Títulos
  • <p>    - Tag utilizada para fazer parágrafos
  • <ul>  - Tag utilizada para fazer listas não ordenadas
  • <ol>  - Tag utilziada para fazer listas ordenadas
  • <li>  - Tag utilizada para fazer linhas nas listas
  • <img> - Tag utilizada para colocar uma imagem

    Elementos não visíveis / parcialmente visíveis na página

  • <title> - Tag utilizada colocar um título na aba
  • <abbr> - Tag utilizada para dar significado a uma sigla
  • <link> - Tag utilziada para fazer linkar um conteúdo CSS externo
  • <li>  - Tag utilizada para fazer linhas nas listas
